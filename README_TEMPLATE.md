# Italy Economic Status Dashboard

## Descrizione

Questa dashboard analizza l'evoluzione della condizione economica del cittadino medio italiano dal 2000 ad oggi.

## Fonti dati

- ISTAT
- Eurostat

## Metodologia

I dati vengono:
1. Estratti tramite script personalizzato
2. Normalizzati in formato JSON
3. Versionati su GitHub
4. Serviti staticamente tramite Vercel

- Salari reali
- Inflazione
