# Italy Economic Status Dashboard

## Obiettivo

Creare una dashboard data journalism che racconti come è cambiata la condizione economica del cittadino medio italiano dal 2000 ad oggi.

Stile: Visual / Storytelling (inspirazione Our World in Data).

Non è un SaaS. È un progetto portfolio.

---

## Indicatori principali (6)

1. Reddito disponibile reale pro-capite
2. Salari reali
3. Inflazione (Indice prezzi al consumo)
4. Tasso di occupazione 15-64
5. Consumi reali delle famiglie
6. Rischio di povertà o esclusione sociale

---

## Filosofia tecnica

- Nessuna API chiamata lato client
- Nessun cron automatico
- Dati aggiornati tramite script on-demand
- JSON statici versionati su GitHub
- Deploy statico su Vercel
